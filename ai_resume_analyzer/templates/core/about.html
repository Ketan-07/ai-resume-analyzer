About ResumeAI

What is this?

AI Resume Analyzer is a production-quality Django web application that uses Google Gemini AI to compare your resume against any job description. It provides a detailed breakdown of your strengths, weaknesses, skills gap, ATS compatibility, and actionable improvement suggestions.

Tech Stack
  • Django 5 — Web framework
  • Google Gemini API — AI analysis engine
  • Bootstrap 5 — Responsive UI
  • SQLite — Database
  • PyPDF2 — PDF text extraction
  • ReportLab — PDF report generation
  • Gunicorn + WhiteNoise — Production serving
Privacy

Your resume text is sent to the Google Gemini API for analysis. Uploaded files are stored locally. We do not share your data with third parties. This tool is intended for personal use and learning.
